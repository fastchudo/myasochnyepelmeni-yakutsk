# МЯСОчные — сайт для покупателей в Якутске

Готовый проект на React + Vite.

## Что можно легко менять

- `src/data/stores.js` — список магазинов
- `src/data/products.js` — товары
- `src/data/contacts.js` — контакты
- `src/data/benefits.js` — преимущества
- `src/data/districts.js` — районы

## Как запустить локально

```bash
npm install
npm run dev
```

## Как опубликовать на Vercel

1. Загрузите весь проект в GitHub.
2. Зайдите в Vercel.
3. Нажмите Add New → Project.
4. Выберите репозиторий `myasochnye-yakutsk`.
5. Нажмите Deploy.

## Как обновлять вместе с ChatGPT

Присылайте изменения в таком виде:

```text
добавь магазин
район: Сайсарский
название: Новый маркет
адрес: Якутск, Лермонтова, 20
время: 09:00–22:00
телефон: +7 (4112) 00-00-00
```

После этого обновляйте только нужный файл в GitHub./
